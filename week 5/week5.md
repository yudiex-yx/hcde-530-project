# Week 5 — Competency claim

I can pull data from a web API with `requests`, pull out the fields I need, and turn them into a pandas DataFrame so each row is one record and each column is one variable. Once it’s loaded, I use `head()` to see what the table actually looks like and `info()` to check column types and how many values are filled in. For analysis I lean on `value_counts()` for categories, simple filters like `df[df["col"] > n]` to zoom in on part of the data, and `groupby()` when I want an average or sum by group. I also run `isnull().sum()` so I know which columns have gaps before I trust averages or comparisons. When I write up what I found, I stick to plain English—I say what the numbers show and what that might mean for the question I asked, without dumping jargon. I’m still learning how far I can generalize from a small sample, but I know how to walk through the steps cleanly from API to answer.
